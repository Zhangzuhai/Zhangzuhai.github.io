spring boot使用mybatis

创建数据库User

创建表userinfo

插入一条数据

创建一个spring boot项目
pom加入依赖

application.properties中配置

写个测试类测试一下是否能连接数据库

控制台正常打印，说明能够成功连接上

项目结构

主类上加上注解@MapperScan(“com.zzuhai.mybatispractice.mapper”)

新建entity对应数据库字段

新建mapper接口，加上@Mapper注解

新建mapper.xml，id和mapper接口中一致，namespace对应mapper接口的路径地址，设置好入参和反参类型，编写sql语句

配置mybatis-config.xml

新建servcie接口和实现类来调用mapper接口里的方法，进行数据库操作

最后通过测试类来测试一下

整合完成
